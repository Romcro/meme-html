Comment écrire du code html

Pour écrire du code HTML vous allez avoir besoin d’un éditeur de texte. Il s’agit tout simplement d’un logiciel permettant d’écrire du code HTML. Vous pourriez utiliser sans problème le bloc note de votre machine mais, les éditeurs de texte vont vous permettre de comprendre plus facilement grâce à leur présentation et de vous y retrouver dans vos lignes de code.
