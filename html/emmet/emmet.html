Les raccourcis EMMET

Si si rappelez vous nous avons téléchargé et installé ce plugin tout à l’heure… Je vous ai dit qu’il allait vous faire gagner du temps et vous rendre beaucoup plus efficace et productif, voici comment:

Utiliser > pour ajouter un élément enfant (child)

li>p puis tab

Utiliser + pour ajouter un élément frère (sibling)

p+a puis tab

Utiliser * pour multiplier les éléments

ul>li*4 puis tab

Utiliser . pour ajouter un nom de classe

section.maclasse puis tab

Utiliser # pour ajouter une identité

section#maclasse puis tab

Utiliser () pour grouper des éléments multiples

header+(main>section*3>article*2>h2+p)+footer puis tab

Utiliser {} pour insérer du texte dans un élément

a {Click Me} puis tab

Ajouter une numérotation avec $

ul>li$*5 puis tab
